# Codex 개인 AI 에이전트 제품 기획서

## 1. 제품 개요

- 제품명: CodexOS (가칭)
- 웹 기반 개인 AI 개발 도구

Codex CLI와 연결하여 다음 기능을 제공한다.

- 코드 생성
- 디버깅 지원
- 구조 설계
- 개발 자동화
- 프로젝트 생성

로컬 환경에서 실행되는 개인 개발 에이전트이다.

## 2. 주요 사용자

### 주 사용자

- 개발자
- 1인 개발자
- 기술 기반 창업자

### 보조 사용자

- AI 파워 유저
- 자동화 개발자

## 3. 핵심 가치

단순 채팅 AI가 아니라, "지속적으로 기억하는 개인 개발 에이전트"를 목표로 한다.

### 제공 기능

- 세션 기억
- 프로젝트 컨텍스트 유지
- 스트리밍 응답
- 로컬 실행

## 4. MVP 기능

### 4.1 채팅 인터페이스

- 메시지 목록
- 입력창
- 스트리밍 응답 표시

### 4.2 Codex CLI 연동

- `stdin` 입력
- `stdout` 스트리밍
- 오류 처리
- 타임아웃 관리

### 4.3 세션 관리

- 여러 세션 지원
- 세션별 기록 저장
- 대화 컨텍스트 유지

#### 저장 방식

- MVP: JSON 파일
- 향후: SQLite / Redis

## 5. 사용자 흐름

1. `/agent` 페이지 접속
2. 세션 생성 또는 선택
3. 메시지 입력
4. `/api/agent` 호출
5. 서버에서 프롬프트 생성
6. Codex 실행
7. 결과 스트리밍
8. 세션 저장

## 6. 기술 스택

### Frontend

- Next.js (App Router)
- React
- Streaming Fetch

### Backend

- Next.js Route Handler
- Node `child_process`
- Agent Layer

### AI 엔진

- Codex CLI

### 저장

- JSON

## 7. 비기능 요구사항

### 성능

- 첫 응답 2초 이내
- 스트리밍 필수

### 안정성

- 60초 타임아웃
- 프로세스 오류 복구

### 보안

- Codex 경로 고정
- 입력 검증
- 임의 명령 실행 금지

## 8. 성공 지표

- 응답 속도
- 오류 발생률
- 세션 수
- 세션당 메시지 수

## 9. 로드맵

### Phase 1

- 채팅 + Codex 연동 + 세션 저장

### Phase 2

- 파일 시스템 접근
- 프로젝트 컨텍스트

### Phase 3

- Git 제어
- 터미널 실행

### Phase 4

- 자동 프로젝트 생성 에이전트

## 10. 최종 비전

CodexOS는 다음으로 발전한다.

- 개인 AI 개발자
- 로컬 개발 환경 관리자
- 자동 코딩 시스템

### 포지셔닝

"개인 AI 엔지니어"
